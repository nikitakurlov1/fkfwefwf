# Доступ к Админ-панели

## Как получить доступ

1. Откройте виджет техподдержки (кнопка чата в правом нижнем углу)
2. В поле ввода сообщения введите: `admin1236`
3. Нажмите отправить
4. Вы будете автоматически перенаправлены на страницу админ-панели

## Особенности

- **Время доступа**: 24 часа с момента входа
- **Автоматический выход**: После истечения 24 часов доступ автоматически закрывается
- **Защита**: Доступ сохраняется в localStorage браузера
- **Адаптивный дизайн**: Полная поддержка мобильных устройств

## Функции админ-панели

### 1. Управление моделями

#### Просмотр моделей
- **Десктоп**: Таблица со всеми данными
- **Мобильный**: Карточки с основной информацией
- Отображение: ID, имя, возраст, город, цена, статус, просмотры, бронирования

#### Добавление модели
1. Нажмите кнопку "Добавить модель"
2. Заполните обязательные поля:
   - Имя
   - Возраст (от 18 лет)
   - Город
   - Цена (₽/час)
3. Заполните дополнительные поля (опционально):
   - Рост, вес, грудь
   - Цвет волос и глаз
   - Национальность
   - Телефон
   - Описание
4. Нажмите "Добавить"

#### Редактирование модели
1. Нажмите кнопку "Редактировать" (иконка карандаша)
2. Измените нужные поля
3. Нажмите "Сохранить"

#### Активация/Деактивация
- Нажмите кнопку с иконкой глаза
- Статус изменится между "Активна" и "Неактивна"
- Неактивные модели не отображаются на сайте

#### Удаление модели
1. Нажмите кнопку "Удалить" (иконка корзины)
2. Подтвердите удаление
3. Модель будет удалена безвозвратно

### 2. Статистика

Отображает ключевые метрики:
- **Всего моделей**: Общее количество в базе
- **Просмотры**: Суммарные просмотры всех профилей
- **Бронирования**: Общее количество бронирований
- **Доход**: Расчетный доход за период

### 3. Настройки

Управление параметрами сайта:
- Название сайта
- Email для уведомлений
- Telegram для поддержки
- Изменение админ-кода

## Мобильная версия

### Навигация
- Нажмите кнопку меню (☰) в левом верхнем углу
- Выберите нужный раздел
- Меню автоматически закроется после выбора

### Управление моделями
- Модели отображаются в виде карточек
- Все действия доступны через кнопки в карточке
- Модальное окно адаптировано под мобильные экраны

### Формы
- Поля расположены вертикально
- Автоматическая прокрутка к активному полю
- Кнопки занимают всю ширину экрана

## Горячие клавиши

- `Esc` - Закрыть модальное окно
- `Ctrl/Cmd + S` - Сохранить форму (в модальном окне)

## Безопасность

⚠️ **Важно**: 
- Не делитесь кодом доступа `admin1236` с посторонними
- Рекомендуется изменить код доступа в настройках после первого входа
- Всегда выходите из админ-панели после завершения работы
- Не оставляйте устройство без присмотра с открытой админкой

## Выход из системы

Нажмите кнопку "Выйти" в нижней части боковой панели для немедленного выхода из админ-панели.

## Технические требования

- Современный браузер (Chrome, Firefox, Safari, Edge)
- JavaScript включен
- Минимальное разрешение экрана: 320px
- Рекомендуемое разрешение: 1024px и выше

## Поддержка

При возникновении проблем:
1. Очистите кэш браузера
2. Проверьте подключение к интернету
3. Попробуйте войти заново через виджет
4. Обратитесь в техподдержку через Telegram
